Proyecto - Grupo 28

Este proyecto forma parte del Trabajo Integrador 2025 de la cátedra Proyecto de Software. Su propósito es desarrollar una aplicación web que permita gestionar, preservar y difundir los sitios históricos del país, centralizando toda la información relevante en una única plataforma.

En esta primera etapa se implementó la aplicación de administración, desarrollada en Python y Flask, conectada a una base de datos PostgreSQL con soporte geoespacial. A través de esta aplicación, los editores y administradores pueden gestionar sitios históricos, moderar reseñas, validar propuestas y administrar usuarios y roles.

En etapas futuras se desarrollará el portal público en Vue.js, que permitirá a cualquier ciudadano explorar los sitios históricos registrados, conocer su historia, dejar reseñas y proponer nuevos lugares de interés cultural.
