Proyecto Integrador 2025

Sitios Históricos · Grupo 28

Plataforma administrativa para preservar, difundir y gestionar el patrimonio cultural del país.

¿De qué se trata?

Este proyecto forma parte del Trabajo Integrador 2025 de la cátedra Proyecto de Software. Construimos una aplicación web que centraliza la información de los sitios históricos, facilitando su administración, difusión y cuidado.

Qué incluye esta etapa

Implementamos la aplicación de administración utilizando Python + Flask sobre PostgreSQL con soporte geoespacial. Desde aquí, los equipos autorizados pueden gestionar sitios, validar propuestas, moderar reseñas y administrar usuarios y permisos.
